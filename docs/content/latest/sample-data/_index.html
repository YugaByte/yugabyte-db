--- title: Sample data linkTitle: Sample data description: Sample data for YugaByte DB image: /images/section_icons/sample-data/database.svg section: REFERENCE menu: latest: identifier: sample-data weight: 2750 --- Here are sample databases that you can use to explore and learn YugaByte DB. All of these are PostgreSQL-compatible, so you can use them to compare between PostgreSQL and YugaByte DB.
Chinook
Explore the sample database for a digital media store.
Northwind
Explore the classic sales data for Northwind Traders.
PgExercises
Use the sample database that powers the PostgreSQL Exercises website to learn SQL and test your knowledge.
SportsDB
Explore sample sports statistics for baseball, football, basketball, ice hockey, and soccer.
